## 01 서비스 소개

> #### 주짓수 대회 자동화 서비스
> 
> "Royale"은 아날로그로 진행되면서 발생하는 불편함들을 자동화한 서비스이다.

---

## 02 주요 기능

✔ 애니메이션 추천

✔ 애니메이션에 대한 정보와 리뷰, 톡톡을 통한 소통공간

✔ 유저가 본 애니메이션을 분석하여 만든 일본어 단어장

✔ 취향이 비슷한 사람들과 소통의 공간, 오픈채팅
